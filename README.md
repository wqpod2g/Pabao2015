# Pabao2015
Game——游戏模块
PabaoSpring——主模块
Picture——图片保存模块

#开发环境
##
JDK 1.8
##
Hibernate 3.6.4
##
Spring 3.2.9
##
Redis-x64-2.8.2103
##
相关jar包都在WebContent/WEB-INF/lib目录里,fork代码后直接导入Eclipse即可

#配置文件
##
config.properties——为主要配置文件，包含公众号id、secret、以及数据库相关连接信息
##
log4j.properties——日志相关配置文件
##
hibernate.cfg.xml——Hibernate主配置文件
##
Menu——公众号菜单
